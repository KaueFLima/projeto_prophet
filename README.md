# Analise de séries temporais com Prophet.

O repositório possui dois arquivos de execução, o primeiro (analise_de_temp.py) faz a analise temporal de um arquivo csv com dados de temperatura e faz um forecast destes dados,
comparando o resultado da projeção com o resultado dos dados completos.

O segundo (varejo.py) faz a analise de dados de uma rede varejista americana, este exemplo foi para demonstrar como o prophet lida com casos de sazonalidade mais extrema.

# Instruções:

Clone ou faça download do repositório.

Instale os requirements.

Rode o arquivo python de sua preferencia.
